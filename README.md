# Sistema de Login e Cadastro

Projeto desenvolvido para praticar HTML, CSS e JavaScript, simulando uma tela de Login e Cadastro de usuários.

## 📋 Funcionalidades

- Tela de Login
- Tela de Cadastro
- Validação de campos obrigatórios
- Interface moderna utilizando Flexbox
- Navegação entre Login e Cadastro

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📁 Estrutura do projeto

```
Projeto/
│
├── login.html
├── cadastro.html
├── loginstyle.css
├── cadastrostyle.css
├── loginscript.js
└── README.md
```

## ▶ Como executar

1. Faça o download do projeto.
2. Abra a pasta no Visual Studio Code.
3. Execute o arquivo `login.html` utilizando o Live Server ou diretamente no navegador.

## 📚 Objetivo

Este projeto foi desenvolvido para praticar:

- Estruturação de páginas HTML
- Estilização com CSS
- Flexbox
- Manipulação do DOM
- Eventos JavaScript
- Validação de formulários
